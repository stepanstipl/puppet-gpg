puppet-gpg
==========

Puppet module for installing and configuring gpg, creating and managing gpg keys and decrypting files and values

Requirements
------------
- Puppet (?version?
- GPGMe >= 1.12

Functionality
-------------
- Install gpg
- Configure gpg
- Create new keys
- Import keys
- Delete keys
- Decrypt encrypted file
- Provide function to decrypt values
